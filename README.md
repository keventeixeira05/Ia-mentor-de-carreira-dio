# IA Mentor de Carreira 

Este projeto foi desenvolvido como parte do desafio da plataforma DIO, no curso **CAIXA - Inteligência Artificial na Prática**.

## Objetivo
Criar uma simulação de um Mentor de Carreira utilizando Inteligência Artificial para auxiliar pessoas interessadas em ingressar ou se reposicionar na área de tecnologia.

## Como funciona
O IA Mentor de Carreira atua de forma conversacional, fazendo perguntas sobre:
- Interesses pessoais
- Habilidades atuais
- Objetivos profissionais

Com base nas respostas, a IA sugere possíveis caminhos na área de tecnologia, como:
- Desenvolvimento de Software
- Análise de Dados
- Suporte Técnico
- Outras áreas de Tech

## Tecnologias Utilizadas
- Inteligência Artificial (ChatGPT)
- Prompt Engineering
- Plataforma DIO

## Exemplo de interação

**IA:** Quais áreas da tecnologia mais chamam sua atenção?  
**Usuário:** Gosto de tecnologia e de ajudar pessoas.  
**IA:** Uma boa opção pode ser a área de suporte técnico ou UX, pois une tecnologia e atendimento ao usuário.

## Conclusão
Este projeto demonstra a aplicação prática da Inteligência Artificial como ferramenta de orientação profissional, tornando o processo de escolha de carreira mais acessível e personalizado.
